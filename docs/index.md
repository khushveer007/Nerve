## Project Overview

Nerve is a role-based university knowledge hub with a React frontend and an Express/PostgreSQL backend in the same repository. The active runtime uses cookie-session auth, JSON APIs, and VPS-oriented deployment automation. The repo also retains an older browser-only data path and Supabase artifacts that are useful as historical context but are not the main runtime flow today.

## Project Structure

This project consists of 2 parts:

### Frontend SPA (frontend)

- **Type:** web
- **Location:** `src`
- **Tech Stack:** React 18, React Router, TanStack React Query, Tailwind CSS, Radix/shadcn-style UI, Vite
- **Entry Point:** `src/main.tsx`

### API and Data Layer (api)

- **Type:** backend
- **Location:** `server`
- **Tech Stack:** Express 4, PostgreSQL, pg, express-session, connect-pg-simple, Zod, Multer
- **Entry Point:** `server/index.ts`

## Cross-Part Integration

The frontend talks to the backend exclusively through `/api` using `src/lib/api.ts`. The backend persists both business data and sessions to PostgreSQL, writes uploads to disk, and serves JSON contracts consumed by the frontend providers and pages.

## Getting Started

### Prerequisites

- Node.js 22.x
- npm
- Docker with Compose plugin
- `.env.local` for local development

### Setup

```bash
npm ci
cp .env.local.example .env.local
```

### Run Locally

```bash
npm run dev:local
```

### Run Tests

```bash
npm test
```
