## Project Overview

Nerve is a role-based knowledge hub for Parul University teams. The active codebase consists of a Vite/React frontend and an Express/PostgreSQL backend in one workspace. The repository also retains Supabase schema and Edge Function assets that document a cloud-hosted variant of the product, but that path is currently disconnected in the live client.

## Project Structure

This project consists of two active parts:

### Web Client (`web-client`)

- **Type:** Web frontend
- **Location:** `src/`
- **Tech Stack:** React 18, React Router, Vite, Tailwind CSS, Radix UI
- **Entry Point:** `src/main.tsx`

### API Server (`api-server`)

- **Type:** Backend API
- **Location:** `server/`
- **Tech Stack:** Express 4, pg, express-session, connect-pg-simple, Zod
- **Entry Point:** `server/index.ts`

## Cross-Part Integration

The browser calls the API through `src/lib/api.ts`, using cookie-based requests to `/api`. Vite proxies those requests in development and Nginx proxies them in production. The API stores both application data and session state in PostgreSQL. Separately, `supabase/` retains an older or future cloud path for auth, data, attachments, and AI generation.

## Getting Started

### Web Client Setup

**Prerequisites:** Node.js 22, npm, API running locally or via Docker

**Install and run:**

```bash
npm ci
npm run dev
```

### API Server Setup

**Prerequisites:** Node.js 22, PostgreSQL or Docker Compose, populated env vars

**Install and run:**

```bash
npm ci
npm run dev:server
```
